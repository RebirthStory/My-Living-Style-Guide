What is a Living Style Guide?

A living style guide is a reference used by web designers and developers to help them understand the look and feel of a website or application. The guide is comprised of elements and patterns that promote a consistent style across a whole project (Hildebrandt 2015).

Why is a living style guide useful to developers and designers?

Web designers and developers use a living style guide to increase their rate of productivity â€“ it allows them to rapidly build features that inherits the already established design of the application or website.

A living style guide also allows new problems to be tackled by using existing patterns in the application, rather than working with nothing to resolve issues like when building applications from the beginning (Lambert 2016).

List of Living Style Guide Tips

  • Keep It Short and Simple (KISS) so that it is easy to read and understand
  • Examples should be used as well to show how things are structured so that consistency is maintained
  • Provide description for elements so that developers and designers know why certain decisions were made for the style
  • Style guides should be updated often (Young 2016).
