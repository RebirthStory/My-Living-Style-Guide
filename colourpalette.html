Colour Palette Page

What colours to use?

Colours used within this website have passed at least WCAG AA standard when running them through the Webaim.org Colour Contrast Checker.

Their use is definitely recommended when creating or modifying webpages on this website because it promotes accessibility - and because of the WCAG AA rating, these colour schemes can also be used for when creating websites for Government Agencies.

The information below contains details in regards to what colours are used, where they are used and their ratings.

Text Colours

  • Text colour used for Header, Navigation and Footer is White (#FFFFFF)
  • Text colour used for Section is Black (#000000)

Header and Footer Colours

  • Background colour used for the Header and Footer is a light purple-type colour (#494E6B)
  • In conjunction with the white-coloured text it has a contrast ratio of 7.45:1
  • JavaScript Reference: Week-5-JS-Event-Listeners

Navigation Bar Colours

Navigation Bar

  • Background colour used for the Navigation Bar is a dark blue-type colour (#192231)
  • In conjunction with the white-coloured text it has a contrast ratio of 15.98:1

Active Tab

  • Background colour used for an active tab on the Navigation Bar is a maroon-type colour (#985E6D)
  • In conjunction with the white-coloured text it has a contrast ratio of 5.05:1

Hovering Over Tabs

  • Background colour used for when hovering over a tab on the Navigation Bar is a red-type colour (#E20004)
  • In conjunction with the white-coloured text it has a contrast ratio of 4.96:1

Section Colours

  • Background colour used for Section is a beige-type colour (#F5F1DE)
  • In conjunction with the black-coloured text it has a contrast ratio of 18.52:1
