Grid Sizes

A grid system that can be used when creating websites is the one shown below that uses 8 flexible columns. There is a gutter between columns that is 30px wide. A border-box is applied so that the border and padding is included in the width of the grid columns. JavaScript Reference: Week-5-JS-Event-Listeners

Grid Gutters

The grid columns have a 15px padding on each side which results in the 30px gutter between columns and a 15px gutter on the grid edges.
